# ⚙️ VocaVault API

RESTful API powering the **VocaVault** English vocabulary learning platform.

---

## 🚀 Features

- User Authentication
- Vocabulary Management
- Categories
- Videos
- Word Forms
- RESTful API
- Token Authentication
- Automatic Deployment

---

## 🛠 Tech Stack

- Laravel 12
- PHP 8
- MySQL

---

## 🌐 Base URL

Production

```
https://api.vocavault.ahmedmagdy.cloud
```

---

## 📦 Installation

```bash
git clone https://github.com/ahmedmagdyoff/vocavault-api.git

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```
